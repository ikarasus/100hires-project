## 🐛 Issues Encountered & Solutions
 
### Issue: Extensions Were Not Visible After Installing Cursor
 
**What happened:**
After downloading Cursor, it launched directly into **Agent View** — and there were no extensions visible anywhere in the interface.
 
**Root Cause:**
Extensions are only accessible from the **Editor View**, not the default Agent View that Cursor opens on first launch.
 
**How I Fixed It:**
 
1. 🔍 Searched YouTube for *"how to install extensions in Cursor IDE"*
2. 📖 Checked the [official Cursor documentation](https://docs.cursor.com)
3. 💡 Discovered that switching to **Editor View** reveals the Extensions panel
4. ✅ Installed the required extensions successfully after switching views
> **Tip for others:** If you can't find extensions after installing Cursor, look for the **Editor View** toggle — extensions live there, not in the Agent View.
